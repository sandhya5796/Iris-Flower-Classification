# Iris Flower Classification using KNN

## Project Overview
This project uses the K-Nearest Neighbors (KNN) algorithm to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps
1. Load Iris dataset
2. Data preprocessing
3. Train-test split
4. KNN model training
5. Model evaluation
6. Prediction

## Results
The model achieved high classification accuracy on the test dataset.
